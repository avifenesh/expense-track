# Technology Stack\n\n## Core Framework & Language\n- **TypeScript:** The primary programming language for type-safe development.\n- **Next.js 16 (App Router):** The core React framework, utilizing Server Components and Server Actions for a modern full-stack architecture.\n\n## Frontend & Styling\n- **React 19:** The underlying UI library.\n- **Tailwind CSS 4:** The utility-first CSS framework for styling.\n- **Lucide React:** Icon library for purposeful iconography.\n- **React Hook Form + Zod:** For efficient form management and robust schema-based validation.\n\n## Backend & Database\n- **Prisma ORM:** The primary interface for database interactions and migrations.\n- **PostgreSQL:** The relational database (tested with Neon).\n- **Node.js 20+:** The runtime environment.\n\n## AI & Intelligence\n- **Amazon Bedrock:** Integration for AI-powered insights and chat functionality.\n- **Vercel AI SDK:** To streamline the integration of AI capabilities.\n\n## Infrastructure & Deployment\n- **Docker:** Used for local database orchestration.\n- **Railway:** The targeted platform for production deployment and hosting.\n\n## Quality Assurance\n- **Vitest:** The primary framework for unit and integration testing.\n- **Playwright:** For end-to-end testing of critical user flows.\n- **ESLint:** For code linting and maintaining standards.
